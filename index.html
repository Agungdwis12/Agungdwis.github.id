About Me

Hallo, saya seorang mahasiswa Informatika yang memiliki ketertarikan mendalam dalam pengembangan web. Selama masa perkuliahan, saya telah mengerjakan berbagai proyek yang memperkaya pengetahuan saya dalam pemrograman dan desain web. Saya percaya bahwa teknologi web memiliki potensi besar untuk menghubungkan orang-orang serta menciptakan pengalaman digital yang menarik. Melalui setiap proyek yang saya selesaikan, saya berfokus untuk menciptakan solusi kreatif yang tidak hanya fungsional, tetapi juga inovatif di dunia digital.

Sebagai seorang pembelajar yang terus berkembang, saya sedang menjalani pendidikan di bidang Ilmu Komputer, dengan fokus pada penguasaan bahasa pemrograman seperti HTML, CSS, dan JavaScript. Saya secara aktif mengikuti kursus daring dan workshop untuk memperdalam pemahaman saya dalam pengembangan web. Dengan tujuan untuk memperkaya portofolio saya, saya berusaha menghasilkan proyek-proyek berkualitas yang dapat mencerminkan kemampuan dan minat saya dalam dunia teknologi digital.

My Project

Membuat web aplikasi kasir menggunakan React dan Vite, dengan JSON sebagai database untuk mengelola transaksi dan produk secara real-time. Aplikasi ini memungkinkan pengelolaan data yang cepat dan efisien dengan antarmuka yang responsif dan modern.

Pengembangan aplikasi CRUD sederhana menggunakan PHP untuk mengelola data secara efisien. Aplikasi ini memungkinkan pengguna untuk menambah, mengedit, dan menghapus data melalui antarmuka yang mudah digunakan.

Membuat Web toko kaos sederhana menggunakan Java, dengan fitur untuk memindahkan detail pembelian ke WhatsApp. Aplikasi ini memudahkan pelanggan untuk melakukan pemesanan dan berkomunikasi langsung dengan penjual melalui WhatsApp.

Membangun model prediksi untuk Indeks Pembangunan Manusia (IPM) dengan menggunakan algoritma regresi linier. Dataset yang digunakan diperoleh dari Kaggle dan berisi berbagai variabel ekonomi yang mempengaruhi tingkat pembangunan manusia di berbagai provinsi.

Proyek pembuatan model 3D kapal menggunakan Blender, dengan fokus pada teknik pemodelan dan tekstur untuk menciptakan desain kapal yang realistis.

Pembuatan tampilan aplikasi Manga Plus menggunakan Figma, dengan desain halaman Creators, History, My shelf, Browse, Updates dan Hot. Alur perpindahan antar halaman dirancang untuk memastikan pengalaman pengguna yang lancar dan intuitif.
